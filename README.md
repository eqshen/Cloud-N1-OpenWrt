# 云编译 N1 OpenWrt 固件

**说明**：
- 本项目使用 Github Actions 下载 [Lean](https://github.com/coolsnowwolf/lede) 的 `Openwrt` 源码仓库，进行云编译。
- 本项目使用定时编译（北京时间每周日下午4点开始自动运行编译）及触发编译（更新 `README.md`、 `script.sh`、 `config.sh`后可开始编译）两种方式。
- 本项目编译固件适配斐讯 N1 盒子，如需刷机，可直接下载 [releases](https://github.com/huangqian8/Cloud-N1-OpenWrt/releases/latest) 内固件。
- 本项目相对源码默认设置做了如下更改：

**增强项**：（**打勾项**默认**编译**入固件；**未打勾项**默认**不编译**入固件。具体功能详见截图。）

**补充说明**：
- 自动打包发布上传Release包的原理是调用github提供的api接口，所以需要配置对应的token。yml中使用了 ${{ secrets.ACTIONS_RELEASE }} 变量，所以需要在项目settings -> secrets and variables -> Actions 创建 Repository secrets，key名称为ACTIONS_RELEASE，值为你自己api对应的token（这个token需要在个人主页设置中创建）。
